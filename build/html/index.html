How To Use This Guide

Start with the published structure, then zoom into a program or course page only when you need detail.

This site maps the current approved calendar structure for SEOS programs and related pathways. Start with the program directory when you want to compare published pathways, then use the course directory to trace where specific courses appear across the SEOS program set.

Draft revisions, consultation notes, and proposal history still belong on the curriculum site. This guide is the baseline view of what is currently published.

Regeneration

  2. Rebuild every graph and every page as static HTML and SVG.
